# 📊 Instagram User Behavior Analysis (SQL Project)

## 📌 Overview
This project analyzes user behavior, engagement patterns, and content performance on an Instagram-like platform using SQL.

The goal is to extract insights that can improve user engagement, content strategy, and marketing decisions.

---

## 🗂️ Dataset Description

The dataset includes:

- users – user information  
- photos – posts created by users  
- likes – engagement through likes  
- comments – user feedback  
- tags – hashtags  
- photo_tags – mapping between posts and hashtags  
- follows – user relationships  

---

## 🎯 Objectives

- Analyze user activity levels  
- Identify top users and influencers  
- Evaluate hashtag performance  
- Understand engagement trends  
- Provide business insights  

---

## 📁 Files in this Repository

- **social-media-analysis.sql** → All SQL queries and solutions  
- **task-questions.docx** → Detailed explanations and answers  
- **social-media-analysis.pptx** → Final presentation with insights  

---

## 📊 Key Insights

- Majority of users are highly active  
- Inactive users present re-engagement opportunities  
- Hashtags like #smile, #beach, and #party drive high engagement  
- Certain users consistently generate high interaction → potential influencers  

---

## 💡 Business Recommendations

- Use high-performing hashtags in campaigns  
- Collaborate with high-engagement users  
- Target inactive users with re-engagement strategies  
- Focus on trending content themes  

---

## 🚀 Outcome

This project demonstrates how SQL can be used to analyze user behavior and generate actionable business insights for social media platforms.
